This application uses Snapper, a tool for managing filesystem snapshots. Snapper is licensed under the GNU General Public License version 2.

### This application communicates with Snapper through

- Command-line interface
- D-Bus API (org.opensuse.Snapper)
- Snapper C++ library (libsnapper)
